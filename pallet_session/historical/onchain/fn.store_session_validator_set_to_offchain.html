Expand description

Store the validator-set associated to the session_index to the off-chain database.

Further processing is then done off-chain side.

Must be called from on-chain, i.e. a call that originates from on_initialize(..) or on_finalization(..). Must be called during the session, which validator-set is to be stored for further off-chain processing. Otherwise the FullIdentification might not be available.
