Expand description

A utility trait to get a session number. This is implemented for MembershipProof below to fetch the session number the given session membership proof is for. It is useful when we need to deal with key owner proofs generically (i.e. just typing against the KeyOwnerProofSystem trait) but still restrict their capabilities.
