“rocksdb.cfstats-no-file-histogram” - returns a multi-line string with general columm family stats per-level over db’s lifetime (“L”), aggregated over db’s lifetime (“Sum”), and aggregated over the interval since the last retrieval (“Int”). It could also be used to return the stats in the format of the map. In this case there will a pair of string to array of double for each level as well as for “Sum”. “Int” stats will not be affected when this form of stats are retrieved.
