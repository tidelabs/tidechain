“rocksdb.compression-ratio-at-level” - returns string containing the compression ratio of data at level , where is an ASCII representation of a level number (e.g., “0”). Here, compression ratio is defined as uncompressed data size / compressed file size. Returns “-1.0” if no open files at level .
